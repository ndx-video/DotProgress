# General — Example progress entry

## Summary

This file demonstrates a **General** Dot Progress entry: work that is meaningful to record but not tied to a roadmap milestone. The filename uses the `G` prefix with a six-digit global counter; frontmatter `milestone` is set to `General` (not a roadmap label).

## Changes

- Added `.progress/G000001.example-general-entry.md` — this example

## Decisions

- General entries use `G{nnnnnn}.{descriptor}.md` — one six-digit number, no per-milestone index segment.
- Frontmatter `milestone` is the literal string `General`, parallel to roadmap entries using `M0 — Bootstrap`.

## Follow-ups

- Use `G` for convention changes, tooling, docs, or other work outside the current ROADMAP milestone.
